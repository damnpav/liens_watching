Lien's Project
Observation of real estate's price. Suppose that objects are banking lien
Project contains web parser of lien objects and models for pricing of objects and detecting anomalies

Folders:

data - folder with some operational data for developing
htmls - folder with examples of html pages for parsing
model - folder for developing of models
venv - folder for virtual environment
notebooks - folder with notebooks for research and predeveloping
parser - folder with parser
